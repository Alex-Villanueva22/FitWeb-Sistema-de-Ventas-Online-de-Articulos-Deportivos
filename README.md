# 🏋️‍♀️ FitWeb: Sistma de Ventas Online de Artículos Deportivos

> Sistema integral de venta de artículos deportivos que destaca por haber sido sometido a un pipeline estricto de pruebas de software (Caja Blanca y Caja Negra) para garantizar su robustez.

---

## 🎯 El Desafío
El comercio electrónico requiere transacciones seguras y una lógica de negocio sin fallos entre los roles de Cliente, Vendedor y Administrador.
* **Problema:** Riesgo de errores en la gestión de carritos, inventarios y permisos de usuario.
* **Solución:** Una aplicación Full Stack (MERN) validada mediante metodologías de pruebas formales para asegurar un 0% de errores críticos en los flujos principales.

## 🛠️ Stack Tecnológico
* **Frontend:** React.js + Vite (Interfaz de usuario dinámica).
* **Backend:** Node.js + Express.
* **Base de Datos:** MySQL (Gestión relacional de usuarios y productos).
* **Gestión de Paquetes:** npm.

## 👤 Gestión de Roles y Permisos
* **Cliente:** Navegación, carrito de compras y gestión de pedidos.
* **Vendedor:** Control de inventario propio y publicación de productos (multimedia y descripción técnica).
* **Administrador:** Auditoría de productos y gestión maestra de usuarios.

## 🧪 Aseguramiento de la Calidad (QA)
Este proyecto no solo es funcional, sino que ha sido auditado mediante:

### ⬛ Pruebas de Caja Negra (Funcionalidad)
* **Análisis de Valores Límite:** Validación de precios y stock.
* **Tablas de Decisión:** Lógica compleja de pedidos y estados de cuenta.
* **Flujo de Casos de Uso:** Verificación del "Happy Path" y flujos alternos.

### ⬜ Pruebas de Caja Blanca (Estructura de Código)
* **Complejidad Ciclomática:** Análisis de la mantenibilidad de las funciones.
* **Cobertura de Decisiones:** Asegurar que cada ramificación del código sea ejecutada.
* **Matriz de Cobertura:** Trazabilidad completa entre requisitos y casos de prueba.
